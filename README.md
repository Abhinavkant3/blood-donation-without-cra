# React Webpack Starter
> React 16.
> Webpack 4.
> webpack-dev-server and a build script.
> Blood donation SPA

## Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start

# Build for production
npm run build
```
